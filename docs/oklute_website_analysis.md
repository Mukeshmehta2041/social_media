## User Flows

### Flow 1: Browse Advertisements
1. User visits homepage or city page
2. Age verification (if first visit)
3. Views list of advertisements
4. Browses VIP Prime Stories (featured ads)
5. Clicks on ad card to view details
6. Views image gallery
7. Contacts advertiser via phone/WhatsApp
8. Direct communication outside platform

### Flow 2: Search for Services
1. User enters search query in header
2. Or applies filters (location, category, etc.)
3. Views filtered results
4. Uses pagination to browse more results
5. Selects ad from results
6. Views full ad details
7. Contacts advertiser

### Flow 3: Post Advertisement
1. User clicks "Post Your Ad"
2. Login/Sign up modal appears
3. Creates account or logs in
4. Fills ad creation form:
   - Selects category
   - Chooses location
   - Adds title and description
   - Uploads multiple images
   - Sets pricing
   - Adds contact info
   - Sets availability
5. Submits ad for publication
6. Ad appears in listings
7. Optionally promotes ad (paid)
8. Receives inquiries directly

### Flow 4: Age Verification
1. User visits site (first time)
2. Age verification modal appears
3. Reads notice about mature content
4. Checks "I confirm that I am 18 or above"
5. Checks "I have reviewed and agree to Terms and Conditions"
6. Clicks "Accept"
7. Gains access to site content
8. Or clicks "Decline" to exit

### Flow 5: Report Inappropriate Content
1. User views advertisement
2. Identifies inappropriate content
3. Clicks "Report" or accesses report feature
4. Selects reason for reporting
5. Submits report
6. Platform moderates content
