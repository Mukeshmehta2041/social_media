# Project Requirements Document

---

## **Core Features**

1. **User Management System**
   * User Registration/Login
   * Profile Management
   * Account Dashboard
   * User Verification System
   * Password Reset/Recovery

2. **Advertisement Management**
   * Post Advertisement (Free)
   * Edit/Update Advertisements
   * Delete Advertisements
   * Advertisement Status Management
   * Image Upload for Ads
   * Ad Promotion Services (Premium – *handled via WhatsApp redirect, not in-app payment*)

3. **Search & Discovery**
   * Advanced Search Functionality
   * Category-based Filtering
   * Location-based Search
   * Keyword Search
   * Search Results Pagination
   * Saved Searches

4. **Geographic Features**
   * City Selection
   * Location-based Listings
   * Multiple City Support
   * Regional Customization

5. **Content Management**
   * Category Management
   * Content Moderation
   * Report/Flag System
   * Content Approval Workflow

6. **Communication System**
   * Contact Forms
   * Messaging System
   * Email Notifications
   * SMS Notifications
   * **WhatsApp Redirect for Payments** – *All payment-related actions will redirect to WhatsApp, where the scanner will be sent. No in-app payment integration will be developed.*

7. **Security Features**
   * Age Verification (if applicable)
   * Content Security
   * Fraud Prevention
   * User Safety Guidelines
