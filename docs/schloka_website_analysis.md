## User Flows

### Flow 1: Browse Advertisements
1. User visits homepage or city page
2. Selects city from dropdown (optional)
3. Views list of advertisements
4. Clicks on ad card to view details
5. Contacts advertiser via phone/WhatsApp
6. Direct communication outside platform

### Flow 2: Post Advertisement
1. User clicks "Post Your Ad"
2. Creates account or logs in
3. Fills ad creation form:
   - Selects category
   - Chooses location
   - Adds description
   - Uploads images
   - Sets pricing
   - Adds contact info
4. Submits ad for publication
5. Ad appears in listings
6. Receives inquiries directly

### Flow 3: Search for Services
1. User navigates to search page
2. Applies filters (location, category, price, age)
3. Views filtered results
4. Saves search (optional)
5. Selects ad from results
6. Contacts advertiser

### Flow 4: Report Inappropriate Content
1. User views advertisement
2. Clicks "Report" button
3. Selects reason for reporting
4. Submits report
5. Platform moderates content
